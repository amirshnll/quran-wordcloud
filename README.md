### Data Cleaning
- remove string punctuation
- remove arabic sings  ّ  ِ   ُ  َ
- replace ك to ک
- replace ي to ی
- replace ة to ه
- replace أ to ا
- replace إ to ا
- replace ؤ to و
- separation of Quranic chapters
